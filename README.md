# Luftqualitaets-Analyse (OpenAQ)

Dieses Projekt analysiert Luftqualitaetsdaten mit OpenAQ-Daten.
Die komplette Analyse, Datenaufbereitung und Visualisierung befindet sich im Notebook
`analyze_openaq_data.ipynb`.


## Projektstruktur

```
.
+- analyze_openaq_data.ipynb
+- data/
|  +- openaq.csv
+- figures/
+- requirements.txt
+- README.md
```

## Datenquellen

- OpenAQ (Rohdaten in `data/openaq.csv`)


## Setup

Empfohlen: Python 3.10+.

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```


## Ausfuehren

1. Notebook `analyze_openaq_data.ipynb` in Jupyter oeffnen.
2. Alle Zellen von oben nach unten ausfuehren.
